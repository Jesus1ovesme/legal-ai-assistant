# Руководство пользователя — AI-помощник юриста

**Адрес:** https://example.com
**Ключ для входа:** `<APP_ACCESS_KEY из .env>`

---

## Что это такое

Инструмент для практикующего юриста. AI-ассистент (Claude Opus 4.7) помогает:

- Искать НПА и судебную практику в первоисточниках (sudact.ru, pravo.gov.ru, consultant.ru, garant.ru)
- Готовить процессуальные документы (иски, претензии, отзывы, ходатайства)
- Анализировать материалы клиента и строить хронологию дела
- Цитировать только реальные нормы и решения — никаких выдуманных номеров

Каждое дело — отдельная папка с собственным Claude-терминалом, документами и историей диалога.

---

## Первый вход

1. Откройте https://example.com/login
2. Введите ключ доступа (выдан выше)
3. Сессия живёт 14 дней. После — снова попросит ключ.

При **5 неудачных попытках** в течение минуты IP блокируется на 15 минут — это защита от подбора.

## Работа с делами

### Создать новое дело

1. Слева кнопка **«+ Новое»**
2. Введите название (например «Иванов vs. ООО Кафе»)
3. Выберите тип дела (ОСАГО / ДТП / Трудовое / Семейное / Наследство / Админ / Уголовное / Госзакупки / Общее)

Тип дела загружает соответствующий стартовый prompt — Claude уже знает что вы работаете по этой категории.

### Переключение между делами

- Клик на папку слева
- Или **Ctrl+K** → введите часть названия → Enter

При переключении ваш предыдущий разговор с Claude сохраняется. Возвращение в папку — продолжение с того же места (через `claude -c`).

### Удаление / переименование

Клик на меню «⋮» возле папки → Переименовать / Удалить. Выделение нескольких через checkbox + кнопка «Удалить (N)».

### Индикаторы статуса

- **🟢 Зелёная мигающая точка** — Claude активен в этой папке прямо сейчас (открыт терминал).
- **🟡 Жёлтая** — есть запущенный Claude, но никто не подключён (idle).
- **Цветная по типу** — нет активной сессии.

## Терминал Claude (главное окно)

### Как работать

Просто пишите по-русски в нижней строке терминала, нажимайте **Enter**.

Примеры:

```
❯ Подготовь претензию в страховую компанию по ОСАГО
  Клиент: Иванов И.И., ДТП 12.10.2025, виновник в Согласии,
  отказ выплаты со ссылкой на скрытые повреждения.
```

```
❯ Найди свежую практику ВС РФ 2024 года по ст. 152 ГК
  по защите чести публичных лиц
```

```
❯ Сохрани полный текст ст. 1064 ГК в файл st-1064-gk.md
  с актуальной редакцией
```

Claude **сам решает** какие инструменты вызвать, какие документы прочитать, что записать в файл. Ничего вручную выбирать не нужно.

### Toolbar над терминалом

| Кнопка | Что делает |
|---|---|
| **📝 Иск** | Готовый prompt для составления иска по фактам в папке |
| **📨 Претензия** | Составить досудебную претензию |
| **🛡 Отзыв** | Отзыв ответчика по ст. 149 ГПК |
| **🗂 Ходатайство** | Процессуальное ходатайство |
| **🎯 Стратегия** | Анализ дела + правовая стратегия |
| **💾 Архив** | Экспортировать диалог в `chat-history.md` |
| **📦 Сжать** | `/compact` — сократить контекст разговора (если он стал длинным) |
| **🧹 Сброс** | `/clear` — обнулить контекст (Claude забывает диалог, но не файлы) |
| **🎙** | Голосовой ввод (Whisper транскрипция) |
| **⛶** | Развернуть на весь экран (F11) |

### Голосовой ввод

Зажмите **🎙**, говорите, отпустите — текст вставится в терминал. Полезно для длинных описаний фабулы дела.

### Прерывание

Если Claude слишком долго думает или пошёл не туда — нажмите **Ctrl+C** в терминале или закройте папку.

## Документы (правая панель)

### Что туда попадает

1. **Загруженные вами файлы** — drag & drop в правую панель или кнопка «📎 Загрузить»
2. **Файлы, созданные Claude** — когда вы просите «сохрани в файл X.md» — файл появляется здесь автоматически

### Поддерживаемые форматы

- PDF, DOCX, DOC, TXT, MD, JPG, PNG, WEBP
- Лимит: 50 МБ на файл
- Дубликаты (одинаковый sha256) отклоняются

### Фильтры

- **Все** · **Рабочие** (.md, .txt — то что Claude генерирует) · **Исходники** (.pdf, .docx — что вы загрузили)

### Действия с файлом

| Кнопка | Что делает |
|---|---|
| **👁 Открыть** | Превью прямо в окне (для .docx — конвертация через mammoth, для .md — форматированный текст, для .pdf — нативно браузером) |
| **⬇ Скачать** | Сохранить на диск |
| **🗑 Удалить** | Удалить с подтверждением (необратимо) |

### Drag & drop файла в Claude

Возьмите карточку файла из правой панели и **перетащите в терминал** — Claude автоматически получит `@<filename>` как ссылку и подгрузит файл в контекст.

### OCR

PDF и изображения автоматически проходят OCR (Tesseract, русский+английский). Статус виден в карточке файла:
- **Готово** (зелёное) — текст извлечён
- **OCR…** (оранжевое) — обрабатывается
- **В очереди** — ждёт OCR worker'а

## Что Claude умеет (MCP инструменты)

Claude имеет 8 встроенных инструментов для юр-работы. Вы их **не вызываете руками** — Claude сам решает когда что использовать.

| Инструмент | Когда срабатывает |
|---|---|
| `find_law` | Когда упоминаете «ГК», «УК», «ОСАГО», «ЗПП» и т.п. — мгновенный поиск в локальной таблице 56 ключевых актов |
| `validate_citation` | Перед вставкой цитаты «ст. 152 ГК РФ» — проверяет номер, выдаёт URL первоисточника |
| `case_timeline` | После анализа документов — извлекает все даты + события в хронологическую таблицу |
| `search_court_practice` | Поиск судпрактики по запросу через sudact.ru (~50М актов) |
| `fetch_court_doc` | Получить полный текст конкретного решения с sudact.ru |
| `search_npa` | Поиск свежих НПА на pravo.gov.ru (через RU прокси) |
| `garant_search` | Через Гарант API (если есть токен) или fallback на search-URL |
| `garant_fetch_doc` | Текст документа из Гаранта (нужен токен) |

Кэш на стороне сервера: повторный одинаковый запрос — мгновенный ответ.

## Безопасность работы

### Что Claude НЕ может

Каждая папка — это **изолированный sandbox**. Claude в вашей папке:

- ❌ Не выйдет за пределы папки (`cd ..`, абсолютные пути — недоступны)
- ❌ Не запустит произвольную shell-команду (Bash полностью отключён)
- ❌ Не прочитает другие папки дел или системные файлы
- ❌ Не утечёт данные на чужие сервера (WebFetch только на 16 разрешённых правовых сайтов)

Полный текст контракта Claude — в `CLAUDE.md` каждой папки.

### Авторизация

- Один ключ доступа. **Никому не передавайте.**
- При утечке — измените `APP_ACCESS_KEY` в `.env` и рестарт `pm2 restart legal-ai-assistant-web`.

### Бэкапы

⚠️ **Бэкапов сейчас нет.** Если случайно удалите папку или диск откажет — данные потеряются. Для важных дел экспортируйте `💾 Архив` (диалог в `chat-history.md`) и скачайте к себе на компьютер.

## Что делать если что-то не работает

### Я не могу войти

- Проверьте ключ — копируйте без пробелов, без переносов.
- Если 5 раз ошиблись — подождите 15 минут.
- Если совсем не работает — ключ мог быть изменён, спросите у того кто настраивал.

### Терминал показывает «disconnected»

Auto-reconnect через 1-8 секунд. Если 30+ секунд — обновите страницу (Ctrl+R).

### Claude отвечает медленно

Это нормально для сложных запросов с MCP-tools и thinking. Реалистичные тайминги:
- Простой вопрос: 5-15 сек
- Поиск НПА + цитирование: 20-40 сек
- Большой иск с 5+ разделами и поиском практики: 1-3 минуты

Можно следить за прогрессом в терминале — Claude показывает что он делает в данный момент.

### Файл не появился в правой панели

- Подождите 15 секунд (poll-интервал).
- Или нажмите «🔄 Обновить» в правой панели — мгновенный re-scan папки.

### Меня отключило / завис диалог

- Сессия Claude хранится на сервере 1 час idle. Возвращение в папку — продолжение.
- Если упало совсем — `/clear` начнёт новую сессию (но потеряет контекст).
